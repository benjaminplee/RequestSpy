RequestSpy

Need a way to check what was sent to a URL? Done.

1... Request

Send a GET or POST request to any URL starting with request/
e.g. click request/try/me

2... Spy

Spy on your requests by going to the same URL but replace request/ with spy/
e.g. click spy/try/me

Features:
GET and POST requests will be saved
The date, time, headers, and request parameters will be saved
Any URL path and file extension can be used
